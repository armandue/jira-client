This is a small project, which serves for generate timesheet of work logs from Jira.

It is composed by a front end (Angular), and a back end (Spring boot).

The back end is responsible for making an HTTP request to Jira with Jira API, processing data, and sending them to the front end. Meantime, the front end is responsible for summarising data and generating timesheet file.
