This is a small project, which serves for generate timesheet of worklogs from Jira app.

It is composed by a back end (supported by App engine), and a front end.

The back end is responsible for making http request to Jira with Jira api, processing data, and sending them to front end. Meantime, the front end is responsible for generate timesheet file, summarise data, and give some stupid words from the summary (It is even not a good try).

(PS: it only works for employees in Ubiest)
